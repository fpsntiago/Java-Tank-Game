Tank Game

Two players, each player has two lives. whichever player runs out of lives first wins

- Controls for player 1
    - Up: W
    - Down: S
    - Left: A
    - Right: D
    - Shoot: SpaceBar

- Controls for player 2
    - Up: O
    - Down: L
    - Left: K
    - Right: ;
    - Shoot: Shift
    
    JAR does not work, but will run through IntelliJ
